Página de Preguntas

  • ¿Que es HTML y cuál es su función en el desarrollo web?

    Lenguaje de hipertexto se utiliza para dar estructura a una página web.

  • ¿Qué son las etiquetas HTML y cómo se utilizan ?

    Las etiquetas HTML son el lenguaje utilizado para estructurar y definir el contenido en un documento HTML.

  • Proporciona ejemplos de algunas de las etiquetas HTML más comunes

    -body: para el contenido.

    -head: para información sobre el docuemento.

    -div: división dentro del contenido.

    -a: para enlaces.

    -strong: para poner el texto en negrita.

    -br: para saltos de lineas.

    -H1...H6: para titulos dentro del contenido.

    -img: para añadir imagenes al documento.

  • ¿Qué es CSS y cómo se utiliza para dar estilo a una página web?

    Hojas de estilo en cascada, CSS se utiliza para dar estilo al contenido de una página web añadiendo elementos de diseño como colores, fuentes y espaciado.

  • ¿Cuáles son las propiedades más importantes de CSS?

    -Font-family: Define la familia tipográfica.

    -Font-size: Define el tamaño de la fuente y el valor se puede escribir en pixels

    -Color: Define el color de la tipografía.

    -Width: Define el ancho de un elemento, el valor se puede escribir en pixels, ems o porcentaje.

    -Max-width o min-width: Definen el ancho máximo o minimo de un elemento.

    -Height: Define el alto de un elemento.

    -Max-height o min-height: Define el alto maximo o minimo de un elemento.

    -Padding: Es la distancia desde el borde de un elemto hasta su contenido.

    -Margin: Es la distancia entre un elemento y otro.

    -Border: Define el borde de un elemento, su color y grosor

    -Background: Define los fondos de un objeto.El fondo puede ser una imagen o un color

  • ¿Explica el modelo de caja (box model) en CSS?

    El modelo de caja CSS es un módulo CSS que define cajas rectangulares, incluyendo sus rellenos y márgenes, que son generadas para los elementos y que se disponen de acuerdo al modelo de formato visual.

  • ¿Qué es el diseño responsivo y cómo se logra con CSS?

    El diseño responsivo funciona mediante el uso de diseños flexibles, imágenes y consultas de medios CSS para ajustar el aspecto de un sitio web en función del tamaño de la pantalla.

  • ¿Qué es JavaScript y cómo se integra en el desarrollo web?

    JavaScript es un lenguaje de programación que los desarrolladores utilizan para hacer páginas web interactivas. Desde actualizar fuentes de redes sociales a mostrar animaciones y mapas interactivos, las funciones de JavaScript pueden mejorar la experiencia del usuario de un sitio web.

  • Explica cómo javaScript interactua con el HTML y CSS a través del DOM(Document Object Model)

    El DOM en JavaScript es uno de los conceptos clave para desarrollar aplicaciones web interactivas. Es la representación estructurada de un documento HTML o XML en forma de árbol, lo que permite a los desarrolladores manipular y gestionar la estructura, el estilo y el contenido de una página web de forma dinámica.

  • ¿Proporciona ejemplos de cómo JavaScript puede hacer que una página web sea interactiva ?

    cuando haces clic en un botón y aparece un mensaje, eso es JavaScript en acción. Puede gestionar todo, desde tareas sencillas, como cambiar el color del texto cuando pasas el cursor sobre él, hasta acciones más complejas, como validar las entradas de un formulario o crear juegos interactivos.

  • ¿Qué es git y por qué es importrante para el desarrollo web?

    Git es un excelente recurso para el desarrollo web, ya que te permite optimizar las actualizaciones en vivo además de proporcionar una copia de los archivos de tu sitio web.

  • Explica cómo Git se usa para el control de versiones en proyectos de desa web

    Si trabajas en un proyecto a lo largo del tiempo, es posible que quieras realizar un seguimiento de los cambios que se realizaron, quién los realizó y cuándo.

  • ¿Qué es GitHub y cómo facilita la colaboración en proyectos web?

    GitHub es un servicio basado en la nube que aloja un sistema de control de versiones (VCS) llamado Git. Éste permite a los desarrolladores colaborar y realizar cambios en proyectos compartidos, a la vez que mantienen un seguimiento detallado de su progreso.
